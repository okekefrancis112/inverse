# Lambda Sparkling Water Bootcamp

An implementation script that calculates the inverse of
2 modulo 2^{64} - 2^{32} + 1 (using the mini-Goldilocks prime).


## How to use?

### Clone this repo

```sh

git clone https://github.com/okekefrancis112/inverse.git

```

### cd into the repo
```sh

cd inverse
```

### run
```sh

cargo build
```

### run
```sh

cargo run
```
